## Use Cases

### Changing case

-  `~` : case 변경 후 오른쪽으로 커서 이동
-  `{Visual}~`: highlighted text case 변경
-  `{Visual}U` : highlighted text uppercase로 변경
-  `{Visual}u` : highlighted text lowercase로 변경
